== Description ==

- Create Zoom In, Zoom Out button on category page.

- Zoom In/Zoom out to change the column of list products.

- Shortcode [c4dwcgz]

This plugin is used in C4D Vision - Responsive  WooCommerce Theme

You can view here http://coffee4dev.com/responsive-woocommerce-theme-c4d-vision/

== Installation ==

<h3>From your WordPress dashboard</h3>

<ol>
<li>Visit ‘Plugins > Add New’</li>
<li>Search for ‘c4d woo category grid product</li>
<li>Activate plugin from your Plugins page.</li>
</ol>
 

<h3>From WordPress.org</h3>

<ol>
<li>Download Woo Category Carousel.</li>
<li>Upload the ‘c4d-woo-category-grid-product directory to your ‘/wp-content/plugins/’ directory, using your favorite method (ftp, sftp, scp, etc…)</li>
<li>Activate plugin from your Plugins page.</li>
</ol>
